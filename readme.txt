Integrates your Videntity.org profile with WordPress...

== Description ==

* Allows you to display elements from your videntity.org profile on Wordpress.
* Allows you to import Videntity friends to WordPress.
* Allows you to import your Videntity profile to WordPress.

== Installation ==

1. Download the ZIP.
1. Upload the contents to /wp-content/plugins/videntity/ on your server
1. Go to Options > Videntity on your Administration panel for import options, or begin adding code to your template.
